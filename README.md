# LegionHub

> Helping every Legion TD player make better decisions.

LegionHub is an open-source competitive knowledge and decision-support platform for **Legion TD**, initially focused on **Team OZE 11.4b** and **Battle.net Ranked 4v4**.

## Our Mission

Competitive Legion TD knowledge is scattered across Discord, videos, replays, community discussions, old guides and player experience.

LegionHub aims to organize this knowledge into a fast, structured and actionable platform.

Our goal is simple:

**Help players make better decisions before, during and after a match.**

## Core Features

### Legion Database

A structured knowledge base for:

- Units and upgrades
- Waves
- Mercenaries
- Abilities
- Attack and armor types
- Unit synergies
- Counters and weaknesses
- Game-version changes

### Legion Coach

The player provides the current game state:

- Game mode
- Current wave
- Gold
- Lumber
- Workers
- Available roll
- Built units

Legion Coach analyzes the situation and recommends actions such as:

- Build a unit
- Upgrade a unit
- Invest in economy
- Hold gold
- Reroll
- Save lumber
- Send mercenaries

Every recommendation should explain **why** it was suggested.

### Future Features

- Combo Finder
- Build Simulator
- Wave Explorer
- Replay Analyzer
- Data-grounded AI Assistant
- Community statistics

## Project Principle

> Accuracy first. If we do not know something, we do not invent it.

LegionHub should clearly distinguish between:

- Verified game data
- Calculated data
- Community consensus
- Expert opinion
- Experimental recommendations
